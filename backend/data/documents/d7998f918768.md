# DeepSeek-V4  
## Towards Highly Efficient Million-Token Context Intelligence

---

# 🧠 Abstract

DeepSeek-V4 is a new generation of large language models designed for extremely long context understanding.

It includes two main models:

- DeepSeek-V4-Pro  
  - Total parameters: 1.6 trillion  
  - Active parameters: 49 billion  

- DeepSeek-V4-Flash  
  - Total parameters: 284 billion  
  - Active parameters: 13 billion  

Both models support up to 1 million tokens context.

---

## 🚀 Key Innovations

### 1. Hybrid Attention Architecture
- Combines:
  - Compressed Sparse Attention (CSA)
  - Heavily Compressed Attention (HCA)
- Goal:
  - Reduce computation cost
  - Enable efficient long-context processing

### 2. Manifold-Constrained Hyper-Connections (mHC)
- Improves residual connections
- Provides:
  - Better stability
  - Improved signal flow

### 3. Muon Optimizer
- Faster convergence
- More stable training

---

## 📊 Training Details

- Trained on 32+ trillion tokens
- Uses pre-training + post-training pipeline
- Improves reasoning, coding, and agent capabilities

---

## ⚡ Efficiency Improvements

Compared to DeepSeek-V3.2:

- ~73% reduction in inference FLOPs
- ~90% reduction in KV cache

Meaning:
- Faster inference
- Lower memory usage

Result:
- Supports 1M token context

# 🧩 Architecture Overview

Based on:
- Transformer
- Mixture-of-Experts (MoE)
- Multi-Token Prediction (MTP)

New additions:
- mHC
- Hybrid Attention (CSA + HCA)
- Muon Optimizer

---

# 🔍 Attention Intuition

Traditional:
- Every token attends to every token

CSA:
- Compress + select top important tokens

HCA:
- Heavier compression for extreme efficiency
